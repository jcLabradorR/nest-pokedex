## Ejecutar en desarrollo

1. Clonar el repositorio
2. ejecutar con npm

````
npm install
````
3. tener nest Cli istalado

```
npm i -g @nestjs/cli
```
4.Levantar la base de datos con docker

```
docker-compose up -d
```
5. Conectar Nest con mongo
* Descargar el paquete
* configurar mongo

````
npm i @nestjs/mongoose mongoose
````

## Stack usado
* MongoDB
* Nest
